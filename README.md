# 決済ナビ（payment-navi）

個店（対面BtoC）のための、**完全中立**なキャッシュレス決済端末・包括事業者の診断サイト（MVP）。

- 広告・優先表示なし。順位は利用者の入力条件のみで機械的に算出
- 6問の診断 →「年間コスト目安」を月間決済額から実額試算
- 主要6社を収録（Square / Airペイ / STORES決済 / stera pack / 楽天ペイ / PayPay）

## 構成

- `index.html` … 単一ファイル・依存なし。ブラウザで開くだけで動作
- 掲載の料率・費用は参考値（`SERVICES` 配列に集約）。契約前に各社公式で要確認

## ローカルで確認

`index.html` をブラウザで開くだけ。
